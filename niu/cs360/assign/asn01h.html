Homework 1 - Binary & Hexadecimal Arithmetic
(50 points)

Write the answers to these questions on a separate sheet of paper. Write neatly and show your work.
Credit ½ point for each part of questions 1-8, one point for each part of question 9.

  1. Convert the following binary numbers to decimal:

    1. 0110
    2. 11001
    3. 100101
    4. 11000001

  2. Convert the following binary numbers to hexadecimal:

    1. 1101
    2. 11100010
    3. 11110010
    4. 0100101111101101

  3. Convert the following hexadecimal numbers to binary:

    1. 7
    2. 29
    3. 1010
    4. FEED

  4. Convert the following hexadecimal numbers to decimal:

    1. 29
    2. BAD
    3. 256
    4. FFF

  5. Convert the following decimal numbers to binary:

    1. 17
    2. 50
    3. 128
    4. 2049

  6. Convert the following decimal numbers to hexadecimal:

    1. 15
    2. 33
    3. 2512
    4. 1010

  7. Perform the following binary arithmetic: 
     a.      0111            b.    111000            c.   1101101
       + 1010                + 111001               + 1010011
       ------                --------               ---------

 d.     10111            e.    100000            f.  10111111
      - 01101                - 000001              - 10101110
      -------                --------              ----------

  8. Perform the following hexadecimal arithmetic: 
     a.       8AC            b.       D79            c.     A3E4C
        + 460                   - B8A                 + 325CA
        -----                   -----                 -------

 d.      FEED            e.     3B03B            f.    111010
        -BEEF                 - 2BCDE                - 101011
        -----                 -------                --------

  9. The integers in the following computations are indicated in hexadecimal, but represent 32-bit two's complement binary numbers. Perform the operations and indicate if overflow occurs and why. 
      
 a.  61676F3D            b.  BC072ACB            c.  E9A9B540
   + A464322E              - EF562AB3              + 53425ACF  
   ----------              ----------              ----------

 d.  E9D5F02B            e.  A464322E            f.  EF562AB3
   - FED8C506              - 70CBFA4D              + BF07ADC0
   ----------              ----------              ----------

 g.  EDCBA987
   + 80000000
   ----------
