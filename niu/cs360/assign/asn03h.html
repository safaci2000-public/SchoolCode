Homework 3 - Dump Reading Assignment 2
(50 points)

This program creates a table, then attempts to process it. You are to run the program (after adding the necessary JCL), then answer the questions below on a separate sheet of paper (not the output from the job). 
DUMP2    CSECT 
         USING DUMP2,15 
         LA    2,TABLE 
         SR    3,3 
         XREAD DATA,80 
*
LOOP1    BM    ENDLOOP1 
         XDECI 4,DATA 
         ST    4,0(2,3) 
         LA    3,4(,3) 
         XREAD DATA,80 
         B     LOOP1
*
ENDLOOP1 SR    3,3 
         LA    7,TABLE 
         LA    5,TABEND
*
LOOP2    CR    2,5 
         BE    ENDLOOP2 
         L     6,0(,2) 
         ST    6,0(,7) 
         L     7,4(,7) 
         LA    2,4(,2) 
         B     LOOP2
*
ENDLOOP2 BR    14 
         LTORG 
DATA     DS    CL80 
TABLE    DC    30F'-1' 
TABEND   DS    0X 
         END    DUMP2 
0 
1 2 
50 
32 24 19 62 
123 456 789 
987 654 321
Use the resulting output to answer the following questions:

  1. What was the interruption code?

  2. What instruction caused the program to abend?
    Why?

  3. What was the condition code at the time of the ABEND?

  4. How many table entries were built?
    How did you figure this?

  5. What is the return address to the calling routine?
    Where did you find this?
    Does your answer really make any sense?

  6. What are the contents of register 7?

  7. Was any object code changed by this program?
    If so, for which instructions?

  8. Explain why the program ABENDed.
