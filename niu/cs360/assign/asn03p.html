CSCI 360 Fall 2001

Program 3 - Decisions
(100 points)

Overview

The law firm of Dewey, Cheatham, and Howe would like you to write a program to produce a report listing the income generated by its staff of lawyers. The input to the program will be an unknown number of records, each containing a 6-digit id number, a 2-digit rank number, the number of hours billed, and the deduction rate separated by spaces (see SAMPLE INPUT).

For each input record you should print a detail line containing: the id number, rank number, number of hours billed, deduction amount (in cents), gross income (in cents), and net income (in cents). After processing all of the input records, print summary lines with the total number of employees and the average net income generated (in cents). (Don’t worry about trying to round to the nearest cent, just use the quotient as the average.)

Use the following formulas in your calculations:

Gross income = Hours billed * hourly rate

Deduction amount = Gross income * deduction rate / 1000

Net income = Gross income – Deduction amount

The hourly rate is determined by the lawyer’s rank in the firm. If the rank number is 11, the rate is $100.00 per hour. If the rank is 22, the rate is $125.00 per hour. If the rank is 33, the rate is $150.00 per hour.

Requirements

  1. Use M, MR, DR, D, and LA as appropriate. Continue to use XREAD, XDECI, XDECO, and XPRNT.

  2. Use BC or BCR with branch masks or extended mnemonics, as determined by your instructor

  3. End your program normally by returning to the operating, using: 

    BCR   B'1111',R14

    NOTE: Your instructor may direct you to end your program in a manner different from a normal return to the caller

  4. You MUST use one literal in this assignment. Your LTORG should be located after the last executable instruction (the BCR B‘1111’,R14 or BR R14).

  5. Use symbolic register notation (R0, R1, ..., R15)

  6. Appropriate page and column headers should be included and your output should start at the top of a new page and be neatly spaced and formatted. (see SAMPLE OUTPUT)

  7. Use a read loop to read in records, scan for all numbers on the input record, perform the necessary processing, build the output line, and write out (double spaced) the output line.

  8. Use line documentation, documentation boxes, TITLE, EJECT, and SPACE as in previous assignments.

  9. The JCL for the input file is: //FT05F001 DD DSN=T90MES1.C360(DATA3),DISP=SHR
