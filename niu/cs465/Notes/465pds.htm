Partitioned Data Set (PDS)

A data set that is made up of different data sets (members).

To create a PDS, the format of the SPACE parameter must be changed to:

 

  FORMAT:  SPACE=(unit,(p,s,b))

A PDS contains a directory of its members. The directory is stored in 256 byte blocks. b is the number of 256 byte blocks.

A directory block contains information on approximately 5 members.

Lets assume that a PDS called Z123456.MYPDS contains a member called MEM1.

 

To reference the PDS:  Z123456.MYPDS

To reference a member of the PDS:  Z123456.MYPDS(MEM1)

NOTES:

  • On an EXEC statement with PGM=program_name, program_name is a member of a partitioned data set.

  • The system first looks for program_name in the library specified on the STEPLIB DD statement(s). If it's not found, the system libraries are checked.

    •  

    //STEPLIB  DD  DSN=MYLIB,...        library made up of one data set


//STEPLIB  DD  DSN=MYLIB,...        concatenated data set
//         DD  DSN=YOURLIB,...        - the three DD statements are
//         DD  DSN=NEWLIB,...           treated as one library
