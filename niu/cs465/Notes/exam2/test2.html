Multiple Choice Questions

Select the correct answer by clicking the appropriate radio button
1.) The KEY(s) used for sorting/merging records using the Cobol Sort and Merge
verbs MUST BE:
a.) in the SD record description.
b.) in the first 4092 bytes of the record.
c.) physically in the same location on each record
d.) All of the above.
e.) None of the above.


2.) When using the Sort and Merge verbs, the programmer:
a.) has to open and close the sort file.
b.) must always code either a RELEASE or a RETURN statement.
c.) must always open and close the Input or Output files.
d.) None of the above.
e.) B and C.


3.) When using the Cobol COPY verb
a.) the code to be copied must be in a PDS.
b.) the programmer can change the copied data names to something else.
c.) the Compiler must be passed the LIB parameter.
d.) All of the above.


4.) What special DD statement is required by the Cobol COPY
verb in the compile step?
a.) COPYLIB
b.) SYSCOPY
c.) SYSLIB
d.) A and B
e.) No special statement is required.


5.) Which of the following is (are) a true statement(s)?
a.) SORT must use an Input or Output procedure.
b.) MERGE must have a Using option.
c.) SORT must use more than one input data set.
d.) MERGE cannot use an Output procedure.
e.) None of the above are true statements.
f.) B and C.


6.) The return code from a Sort or Merge operation can be found:
a.) in the SORT-RETURN-CODE.
b.) in the RETURN-CODE.
c.) in the SORT-RETURN.
d.) All of the above.
e.) None of the above.


7.) When working with Sort/Merge, which of the following is
equivalent to a READ ststement?
a.) RELEASE SORT-WORK-FILE AT END MOVE 'Y' TO EOF-FLAG.
b.) RETURN SORT-WORK-FILE AT END MOVE 'Y' TO EOF-FLAG.
c.) RETURN SORT-WORK-FILE FROM WS-RECORD.
d.) B and C.
e.) None of the above.


8.) In what part of a Cobol Sort/Merge program must at least
one RELEASE statement be coded?
a.) SORT Input procedure.
b.) SORT Output procedure.
c.) MERGE Output procedure.
d.) MERGE Input procedure.
e.) A and D.


9.) Which of the following pairs of options CANNOT be coded
in the same SORT statement?
a.) ON ASCENDING KEY...and ON DESCENDING KEY...
b.) INPUT PROCEDURE IS...and OUTPUT PROCEDURE IS....
c.) INPUT PROCEDURE IS...and USING...
d.) INPUT PROCEDURE IS...and GIVING...
e.) All of these are legal combinations.


10.) What will happen if a program expects parameters from the EXEC card
and the PARM= was omitted from the EXEC card?
a.) It would cause a JCL error.
b.) It would cause a Compiler error.
c.) The Loader/Linkage Editor would generate a parameters missing message.
d.) The program would not execute.
e.) None of the above.


11.) When Cobol calls an assembler program:
a.) Standard entry conventions are observed.
b.) There MUST be a DSECT to receive the parameters in the assembler program.
c.) R1 contains the address of the parameter list.
d.) None of the above
e.) A and C.


12.) What Cobol termination statement causes a return to an invoking program or,
in the absence of an invoking program, causes a return to the O.S.?
a.) STOP RUN.
b.) GOBACK.
c.) RETURN.
d.) None of the above
e.) All of the above.


13.) Which of the following is true of the Cobol linkage section?

b.) It is the Cobol version of an Assembler parameter list.
c.) It must always contain a Length field in the parameter list.
d.) None of the above.
e.) All of the above.


14.) A Cobol program is going to use the SORT verb with the
USING option (ddname INSTUFF) and the OUTPUT PROCEDURE option.

Which of the following ddname(s) must the Cobol program open explicitly?
a.) INSTUFF DD
b.) PRINT DD
c.) SORTWK01 DD
