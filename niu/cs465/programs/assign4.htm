Write the JCL to compile and execute (using the Loader) a Cobol source program that is named T90RAZ1.ASS4CODE. The source program uses the input data set named T90RAZ1.ASS4DATA with the DD card name of MYINPUT. Both data sets are on ACA301. The source program writes its output to the printer with DD card name MYOUTPUT.

The source program will abend when it is executed. You may use any compiler or run-time options that you want to enable you to debug the program and to answer the questions that appear below.  You may only run one job, but you can use whatever compiler or run time options you want.

On your JOB printout, LOCATE, LABEL, and HIGHLIGHT the information (file, instruction, DCB information, etc.) that you use to answer the questions.

You are to work ALONE on this assignment. Your answers must be complete and must be NEATLY written (preferably typed) on 8 1/2 by 11 paper. Illegible answers will not be given any credit.

1.) (4) What is the entry point of the Cobol program? List at least two places where this information can be found.

3.) (4) What is the absolute address of the instruction that caused the abend? How did you find this information? Show any
calculations.

4.) (4) What is the relative address of the next instruction that would have executed had the program not abended? Show your

5.) (4) What is the relative address of the instruction that caused the abend? Show your calculations.

10.) (5) Correct the Cobol source code so the abend can be avoided in future runs. Write your correction(s) on the compiler produced source code. Do not change anything in the Working-Storage section.

12.) (4) What are the contents of FIELD-3 at the time of abend? Show at least 2 ways to find this information.
