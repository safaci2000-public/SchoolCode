CSCI 465 In Class Debugging Assignment (0 points)

Spring 2002

This is a no-credit learning assignment. You are to write the JCL to compile and execute a Cobol program that has already been written. The program will compile, with informational warnings, and will abend when executed.

All parts of this assignment use the LE370 Compiler and Loader. ALWAYS INCLUDE MAP AND LET AS LOADER OPTIONS. The program source code is contained in the file with DSN=T90RAZ1.PRACCODE and the program's input data is in the file with DSN=T90RAZ1.PRACDATA. Both files are on ACA301. Assign the input file data to a DD card with the name INPUT. The program writes its output to the printer via DD card name OUTPUT.

 

JOB NUMBER 1.

*************

Pass the following parameters to the Compiler.

OFFSET, XREF, FLAG(I,I), and APOST

 

JOB NUMBER 2.

*************

Pass the following parameters to the Compiler.

LIST, VBREF, FLAG(W,W), APOST, and TEST(ALL,SYM)

 

JOB NUMBER 3.

*************

Pass the following parameters to the Compiler.

MAP, TEST, and APOST

Pass the Run-time parameter of TERMTHDACT(DUMP).

To pass a run-time parameter when using the Loader, the format is as follows:

EXEC PGM=LOADER,PARM='MAP,LET//TERMTHDACT(DUMP)'

 

When instructed, BRING ALL 3 JOBS TO CLASS. We will discuss each JOB in detail. Without the output, you will be unable to follow the discussion.
